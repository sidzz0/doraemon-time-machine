# 哆啦A梦の时光机

一个哆啦A梦主题的网页小玩具：选择穿越时间，点击开始，然后真的等待对应时间，最后弹出“穿越成功”。

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

构建产物会输出到 `dist/`，可以直接部署到 GitHub Pages、Netlify 等静态托管平台。

## GitHub Pages

仓库推送到 GitHub 后，在仓库 Settings -> Pages 里选择 GitHub Actions。之后推送到 `main` 或 `master` 分支会自动构建并发布。
